# Epstein Files Downloader
Epstein Files Downloader is a specialized utility for scraping and organizing DOJ disclosures into categorized datasets.

> [!CAUTION]
> This tool is intended for research purposes. Ensure you have a stable internet connection for large PDF downloads.

## Installation

### Windows
1. Download the latest **windows** release build and extract the files.
2. Open CMD/Powershell inside directory of **windows** folder.
3. Run start.bat.

### Linux | MACOS
1. Download the latest **linux** release build and extract the files.
2. Open terminal inside directory of **linux** folder.
3. Run `chmod +x start.sh && ./start.sh`
